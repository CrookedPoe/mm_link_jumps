# mm_link_jumps

This will port the two additional jumping animations from Majora's Mask to Ocarina of Time. This can be applied to any clean Debug ROM.

The patch is a .txt file which can be applied to your ROM with CloudMax's [Rom Patcher](https://cloudmodding.com/app/rompatcher).
